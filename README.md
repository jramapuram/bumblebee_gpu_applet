#bumblebee_gpu_applet
A simple tool that can be used with i3blocks in order to provide GPU usage / GPU memory usage and GPU temp for NVIDIA cards using bumblebee

## Usage
```conf
[script]
command=<PATH>/bumblebee_gpu_applet
interval=10
```
